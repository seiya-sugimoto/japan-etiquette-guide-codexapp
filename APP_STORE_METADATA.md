## Suggested Store Copy

### Short description

Calm etiquette guidance for first-time visitors to Japan.

### Longer description

Japan Etiquette Guide helps first-time visitors move through everyday situations in Japan with more confidence.

It is not a sightseeing guide. It is a practical mobile guide to common etiquette, shared-space behavior, and the cultural context behind it.

The app covers situations such as:

- trains, buses, taxis, and airport transfers
- restaurants, cafes, izakaya, bars, and food courts
- shrines, temples, onsen, hotels, and shared spaces
- shopping, museums, festivals, scenic spots, and more

Current strengths:

- 69 etiquette scenes
- 9 supported languages
- calm, easy-to-scan mobile layout
- practical advice with cultural context
